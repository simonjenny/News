# B65 — NewsPWA

## Project Summary

**B65** is a personal tech/IT news aggregator PWA built with Vue 3, TypeScript, Pinia, Vue Router, Tailwind CSS 4, and Vite. It fetches RSS feeds through a backend CORS proxy, aggregates articles, and displays them as a card-based newsboard. The app is installable as a PWA (Service Worker via Workbox).

UI copy is in **German**. The displayed app name is **B65** — not "News".

## Architecture Rules

1. **Never fetch RSS feeds directly** — always go through the CORS proxy at `VITE_API_BASE_URL?url=<encoded>`.
2. **All state lives in the Pinia store** (`newsStore.ts`) — never write to `localStorage` directly from components.
3. **Strict TypeScript** — every new symbol must be typed; use `NewsItem` from `src/types/news.ts` for article objects.
4. **Vue SFCs** use `<script setup lang="ts">` — no Options API, no class components.
5. **Tailwind only** for styling — avoid custom CSS unless no Tailwind utility exists.
6. **`@/` alias** maps to `src/` — always use it for imports (e.g. `@/stores/newsStore`).

## Environment Variables

| Variable | Purpose |
|---|---|
| `VITE_API_BASE_URL` | Base URL of the RSS proxy (e.g. `https://www.b65.ch/api`) |
| `RSYNC_DEST` | rsync deploy target |

Both `.env` and `.env.development` are gitignored. Never commit them.

## Commands

```sh
npm run dev          # Dev server → http://localhost:5173
npm run build        # Type-check + Vite build → dist/
npm run preview      # Serve dist/ locally
```

## Adding Features

### New RSS feed
Edit `src/config/feeds.ts` — add `{ id, name, url, color }`.

### New page
1. Create view in `src/views/`.
2. Add route in `src/router/index.ts`.
3. Link from `AppHeader.vue` or `App.vue`.

## Testing

No tests exist yet. Use **Vitest** + **Vue Test Utils** when adding tests.

## Do Not Touch

- `dist/` — gitignored, regenerated by build.
- `.env` / `.env.development` — gitignored, contain secrets.
- Scaffold leftovers (`HelloWorld.vue`, `TheWelcome.vue`, `WelcomeItem.vue`, `src/components/icons/`) — unused, safe to delete.
